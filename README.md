# pengumuman_kelulusan

A new Flutter project for graduation announcements.

## Prasyarat (Setup PC/Laptop Baru)

Sebelum menjalankan proyek ini di laptop atau PC baru, pastikan Anda telah menginstal komponen berikut:

1.  **Flutter SDK**: [Unduh dan Instal Flutter](https://docs.flutter.dev/get-started/install)
2.  **Java Development Kit (JDK)**: Pastikan JDK terinstal (diperlukan untuk Android).
3.  **Android Studio** atau **VS Code** dengan ekstensi Flutter dan Dart.
4.  **Git**: Untuk mengelola repositori.

## Panduan Konfigurasi

Ikuti langkah-langkah berikut untuk mengonfigurasi proyek di perangkat baru:

1.  **Clone Repositori**:
    ```bash
    git clone [url-repositori]
    cd pengumuman_kelulusan
    ```

2.  **Instal Dependensi**:
    Jalankan perintah berikut di terminal:
    ```bash
    flutter pub get
    ```

3.  **Konfigurasi Lokal (Opsional)**:
    Jika proyek menggunakan file konfigurasi seperti `.env` atau `local.properties`, pastikan untuk menyalin atau membuatnya berdasarkan template yang disediakan (misal: `.env.example`).

4.  **Menjalankan Proyek**:
    Pastikan emulator atau perangkat fisik terhubung, lalu jalankan:
    ```bash
    flutter run
    ```
